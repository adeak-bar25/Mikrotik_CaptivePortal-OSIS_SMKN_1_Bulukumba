# Captive Portal OSIS/MPK SMKN 1 Bulukumba

## Deskripsi
Repositori ini berisi laman Captive Portal/Login Hotspot untuk router Mikrotik yang digunakan oleh OSIS SMKN 1 Bulukumba.
